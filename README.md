# OIBSIP_WebDevelopment_Task1_TemperatureConverter

## project Title
Temperature Converter Website

## Objective
The objective of this project is to create a temperature converter website is to provide a simple, fast, and user-friendly web application that allows users to convert temperature values accurately between different measurement units.

## Features
Convert between:
Convert between Celsius, Fahrenheit, and Kelvin
- Automatically converts to the next unit in rotation
- Rounds results to 2 decimal places
- Responsive design (mobile-friendly)
- Smooth button and result animations
- Accessible (ARIA labels + keyboard support)

## Technologies Used
- HTML5 – Structure
- CSS3 – Styling, animations, responsiveness
- JavaScript (ES6) – Conversion logic and DOM manipulation

 ## How It Works
 **Conversion Logic**
- Converts the input value to Celsius first
- Converts from Celsius to the next unit in the cycle:
- Celsius → Fahrenheit → Kelvin → Celsius

## Formula Used
**Celsius to Fahrenheit**
-(°C × 9/5) + 32

**Fahrenheit to Celsius**
-(°F − 32) × 5/9

**Celsius to Kelvin**
-°C + 273.15

**Kelvin to Celsius**
-K − 273.15

## Responsive Design
- Fully responsive layout
- Mobile-friendly interface
- Adaptive input stacking for smaller screens
- Dark mode support using prefers-color-scheme

## Run Locally
- Download or clone the project.
- Open index.html in your browser.
- Enter a temperature value.
- Select the input unit.
- Click Convert or press Enter.
